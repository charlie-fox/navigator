# 个人导航页面

一个简洁实用的浏览器主页，集成了常用网站链接和搜索功能。

## 功能特点

- **搜索功能**：支持 Google 和 Bing 搜索引擎切换
- **分类导航**：按功能分类整理常用网站链接
- **响应式设计**：适配桌面和移动设备
- **悬停提示**：每个链接都有详细的功能说明
- **简洁美观**：现代化的界面设计，使用舒适

## 包含的网站分类

- 🔍 **搜索引擎**：Google、必应、百度、地图服务等
- 📧 **办公工具**：邮箱、文档协作、云存储等
- 💬 **社交媒体**：微信、微博、知乎、豆瓣等
- 🤖 **AI 工具**：ChatGPT、Gemini、各种AI助手等
- 🛠️ **开发工具**：GitHub、设计工具、笔记应用等
- 🏠 **生活服务**：购物、出行、支付平台等
- 🎬 **娱乐影音**：视频网站、音乐平台等

## 使用方法

1. 将 `navigator.html` 文件保存到本地
2. 在浏览器中打开该文件
3. 将其设置为浏览器主页即可

## 自定义

你可以根据个人需求修改 HTML 文件中的链接，添加或删除网站，调整分类等。
